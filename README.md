# Student Dashboard Demo

Spring Boot + Thymeleaf demo application for managing:
- Students
- Teachers
- Lectures
- Exams

## Technologies
- Java 17
- Spring Boot
- Spring Data JPA
- Thymeleaf
- H2 Database
- Bootstrap 5

## Features
- MVC architecture
- Entity relationships
- Thymeleaf templates
- Navigation menu
- Sample relational data
- H2 in-memory database

## Run Application

```bash
mvn spring-boot:run
